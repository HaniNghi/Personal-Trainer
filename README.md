# 🏋️ Personal Trainer App

A modern **Personal Trainer Management Web Application** built with **React + TypeScript**.

---

## Features

- 👤 Manage customers (add, edit, delete)
- 🏃 Track training sessions
- 📊 Visualize statistics with charts
- 📅 Calendar view for trainings
- 🔔 Toast notifications
- 📱 Responsive UI

---

## 🧰 Tech Stack

### Frontend
- React
- TypeScript
- Vite

### UI & Styling
- Material UI (MUI)
- MUI Icons

### Data Visualization
- @mui/x-charts

### Data Grid
- @mui/x-data-grid

### Date & Time
- dayjs
- moment

### Utilities
- lodash

### Notifications
- react-toastify
